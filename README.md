# 我的小栈 | My Little Post

个人博客网站，纯静态 HTML 构建，托管于 GitHub Pages。

## 主题分类

- 🌱 生活 - 日常记录与生活感悟
- 💻 技术 - 技术分享与学习笔记
- 🎨 创意 - 灵感迸发与创意实践
- ✍️ 随笔 - 随想随写与思考记录
- 🚀 项目 - 项目展示与作品分享

## 技术栈

- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 响应式设计
- 暗/亮模式切换
- 中英双语支持
- GitHub Pages 托管

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 部署

推送到 `main` 分支后，GitHub Actions 自动部署到 GitHub Pages。
